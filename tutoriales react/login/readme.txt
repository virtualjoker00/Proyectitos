Este es mi proyecto para la prueba técnica de React Native. 
Las funcionalidades incluyen:
-Registro de cuenta
-Inicio de sesión
-Formulario con elementos de input de texto, input numérico, checkbox, switch, slider y vista de mapa con coordenadas
-Vista de elementos guardados en memoria local
*Hay vistas extras con funcionalidades no implementadas, pero que serían necesarias si la aplicación se utilizara de forma online 
y considerando el uso del correo electrónico (cambiar contraseña, confirmar email luego de crear una cuenta). Al no estar implementadas
sus funciones, solo oprima confirmar/continuar para ir a la pantalla de registros.
